# Epic Quest - Quick Start Guide

Get up and running in 5 minutes!

## Installation (30 seconds)

```bash
# Clone the repository
git clone <repo-url>
cd game-2

# Run the game (no dependencies needed!)
python main.py
```

That's it! The game uses only Python standard library.

## First Game (3 minutes)

### 1. Create Your Character (1 min)

1. Choose "New Game" from main menu
2. Enter your character name
3. **Pick a class** (Recommended for beginners: **Warrior** or **Paladin**)
4. **Allocate starting points**: Put 2-3 points in your main stat

**Quick Class Guide:**
- **Warrior**: High HP, good damage ‚Üí Put points in Strength
- **Mage**: Fast leveling ‚Üí Put points in Intelligence
- **Rogue**: High dodge, crits ‚Üí Put points in Agility
- **Paladin**: Balanced ‚Üí Spread points evenly
- **Ranger**: Good at exploration ‚Üí Put points in Agility

### 2. First Steps (1 min)

From the main menu:
1. Choose "View Character" to see your stats
2. Choose "Inventory" to see starting items
3. Choose "Quests" ‚Üí "View Available Quests"
4. Accept "First Steps" quest (kill 3 slimes)

### 3. First Battle (1 min)

1. Choose "Travel" ‚Üí "Sunny Meadow"
2. Choose "Search for Enemies"
3. Fight a Slime:
   - Choose "Attack" each turn
   - If HP drops below 30%, use a Health Potion
   - Victory! You gained XP and gold

## Essential Controls

### Main Menu Options
- **Character**: View/manage stats
- **Inventory**: Items and equipment
- **Travel**: Move between locations
- **Search for Enemies**: Start a battle
- **Explore**: Find treasures
- **Quests**: View and complete quests
- **Shop**: Buy/sell items (in towns)
- **Rest**: Restore HP (costs gold in town, free but risky elsewhere)
- **Save**: Save your progress

### Combat Options
- **Attack**: Deal damage (default action)
- **Defend**: Reduce damage next turn (use when low HP)
- **Use Item**: Heal or buff
- **Flee**: Escape (success not guaranteed)

## Troubleshooting

**Game won't start?**
- Ensure Python 3.8+ is installed
- Run: `python --version` to check
- Make sure you're in the game-2 directory

**Lost save file?**
- Check the `saves/` folder
- Saves are named `<name>_save.json`

**Game too hard?**
- Level up more before boss fights
- Buy better equipment
- Keep HP above 50%
- Stock more potions

**Game too easy?**
- Try harder locations
- Fight higher level enemies
- Try a glass cannon build (Mage/Rogue)
